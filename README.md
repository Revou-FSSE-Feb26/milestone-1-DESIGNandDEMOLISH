# Alexander Portfolio
---
This is my personal portfolio website built using HTML and Tailwind CSS.
The purpose of this project is to practice creating responsive layouts, using utility-first CSS, and structuring a multi-section webpage.

## Features
	•	Responsive navigation bar with mobile menu toggle
	•	Smooth scrolling between sections
	•	Hero section with background image
	•	About Me section with profile image and description
	•	Project showcase using grid layout
	•	Contact form with basic validation
	•	Footer with contact details

## Sections

### Home

A hero section that introduces me as a developer with a short description and background image.

### About Me

Provides information about who I am, what I’m learning, and my goals in software engineering.

### Projects

Displays a grid-based layout for projects.
Currently, this section contains placeholders that will be replaced with real projects as I continue learning.

### Contact

Includes a contact form where users can enter their name, email, and message.

### Footer

Contains additional contact information such as phone number and email.

## Technologies Used
	•	HTML5
	•	Tailwind CSS (via CDN)
	•	Google Fonts (Noto Serif)

## How to Run the Project
	1.	Clone this repository
	2.	Open the project folder
	3.	Open index.html in your browser
